Detailed Description

Klasa modeluje interfejs wtyczki.

Klasa modeluje listę interfejs biblioteki dynamicznie konsolidowanej, która stanowi wtyczkę dla programu głównego, który umożliwia egzekujcję poleceń. Wspomniana wtyczkę zawiera implementację nowego polecenia.
